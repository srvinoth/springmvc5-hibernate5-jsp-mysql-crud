# springmvc5-hibernate5-jsp-mysql-crud
Build Spring MVC for a simple CURD application using Spring MVC 5, JSP,Mysql and Hibernate5.


**Create Mysql database**
```bash
create database emp_db
```

## Running the program
To run the application:
```
mvn clean install
```

## Run Tomcat Server
```
tomcat7:run
```
